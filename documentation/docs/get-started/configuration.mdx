# Configuration Environment Files

The `env` directory contains configuration files for running the app in
different environments.

Each file within represents the environment:

```bash
env
├── brand1
│   ├── dev
│   ├── live
│   └── prod
└── brand2
    ├── dev
    ├── live
    └── prod
```

The mocked values for use in jest tests are in
`__mocks__/react-native-config.js`. If the config structure is changed, remember
to update this file.
