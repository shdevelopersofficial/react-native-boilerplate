# Requirements {#requirements}

This react native boilerplate is built with the following technologies:

- React Native Cli

# Getting Started

- Set up your development environment by following the
  [React Native docs](https://reactnative.dev/docs/environment-setup).

- Install the correct version of CocoaPods by installing
  [podenv](https://github.com/kylef-archive/podenv) and running
  `npm run install:pod`.

- Install the correct version of node.js by installing
  [nvm](https://github.com/creationix/nvm) and running `nvm install` in the
  project root.

- Install the correct version of npm by running `npm run install:npm`.

- Install the node modules by running `npm install` and wait for everything to
  be built successfully.
