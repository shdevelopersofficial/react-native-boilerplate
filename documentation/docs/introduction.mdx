#### React Native Packages

- [Reanimated 3.0](https://blog.swmansion.com/releasing-reanimated-3-0-17fab4cb2394)
  Supported.
- [FastImage](https://github.com/DylanVann/react-native-fast-image) Package is
  added.
- [Async Storage](https://react-native-async-storage.github.io/async-storage/docs/install/)
  is added.
- React native [SVG](https://www.npmjs.com/package/react-native-svg) support.
- [lottie](https://www.npmjs.com/package/lottie-react-native) animation support.
- [Redux toolkit](https://redux-toolkit.js.org/) support.
- Native Fils system access.
  [react native fs](https://www.npmjs.com/package/react-native-fs) added.
- [react-native-device-info](https://www.npmjs.com/package/react-native-device-info)
  for device details.
- [Vector Icon](https://www.npmjs.com/package/react-native-vector-icons) pack
  added.
- [patch-package](https://www.npmjs.com/package/patch-package) is added. you can
  modify packages anytime with your choice.
